We have invested significant time and engineering effort into building this protocol. Our long-term vision is for this codebase to become a fully open-source public good for the Internet Computer Protocol (ICP) ecosystem.

To balance innovation protection during the critical early growth phase with our commitment to openness, this repository is released under the **Business Source License 1.1 (BSL 1.1).** It will automatically convert to the **Apache License 2.0** on **July 1, 2029.**

### What This Means for Developers
**Fully Transparent & Auditable** — The complete source code is public. You can freely inspect, audit, build, and test it locally.

**Broad Ecosystem Integration** — You are explicitly encouraged to build on top of the protocol. The license permits compiling, testing, developing applications, making cross-canister calls, integrating frontends, creating developer tools, and building companion applications that interact with our officially deployed canisters on the ICP network.

**Future-Proof Open Source** — On July 1, 2029, the license converts permanently to Apache 2.0 with no further restrictions.

### Why BSL 1.1?
This model allows us to open the source code immediately while preventing direct commercial cloning that could undermine the project's sustainability and network effects. It gives the community full visibility and integration rights today, with a guaranteed transition to a standard open source license in the near future.

We believe this creates the healthiest environment for long-term growth — supporting serious builders while protecting the original investment.
